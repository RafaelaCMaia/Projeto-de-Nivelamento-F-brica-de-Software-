# EcoDescarte

EcoDescarte é um sistema desenvolvido com o objetivo de ajudar cidadãos a realizar o descarte correto de resíduos urbanos.

A aplicação permite que usuários encontrem pontos de descarte, solicitem coleta de objetos de grande porte e registrem denúncias de descarte irregular.

## Objetivo do Projeto

O projeto busca facilitar o acesso às informações sobre descarte correto de resíduos e incentivar práticas mais sustentáveis nas cidades.

## Funcionalidades

- Consultar pontos de descarte
- Buscar locais por tipo de material
- Solicitar coleta de itens grandes (móveis, eletrodomésticos, etc.)
- Registrar denúncias de descarte irregular
- Enviar imagem da denúncia
- Receber confirmação da solicitação

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Figma (prototipação das telas)
